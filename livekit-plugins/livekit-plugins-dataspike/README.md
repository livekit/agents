# Dataspike deepfake detection plugin for LiveKit Agents

This plugin adds support for real-time deepfake detection using the [Dataspike](https://dataspike.io/) within LiveKit Agents.
It enables detection of synthetic or manipulated media during live or recorded video streams.


## Installation

```bash
pip install livekit-plugins-dataspike
```

## Pre-requisites

You'll need an API key from Dataspike. It can be set as an environment variable: `DATASPIKE_API_KEY`
