# LiteLLM plugin for LiveKit Agents

Support for LiteLLM AI Gateway in LiveKit Agents.

## Installation

```bash
pip install livekit-plugins-litellm
```

## Pre-requisites

LiteLLM supports 100+ LLM providers. Ensure you have the appropriate API keys set as environment variables for the provider you choose (e.g. `OPENAI_API_KEY`, `ANTHROPIC_API_KEY`, etc.).
