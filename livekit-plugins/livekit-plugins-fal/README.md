# LiveKit Plugins Fal

Image generation with [Fal.ai](https://www.fal.ai/)'s [SDXL](https://www.fal.ai/docs/fast-sdxl) API.

## Installation

```bash
pip install livekit-plugins-fal
```

## Pre-requisites

You'll need a Fal API key and secret. They can be set as environment variables:

- FAL_KEY_ID
- FAL_KEY_SECRET
